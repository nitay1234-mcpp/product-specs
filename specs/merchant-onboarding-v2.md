# Merchant Onboarding v2

## Problem
Current onboarding takes 3-5 days due to manual KYB review.

## Goal
Reduce to under 24 hours using automated KYB.

## Acceptance criteria
- [ ] Merchant can submit KYB docs via portal
- [ ] Automated verification in < 1 hour for 80% of cases
- [ ] Manual review queue for edge cases
- [ ] Email notification on approval/rejection
